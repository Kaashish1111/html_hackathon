Lecture 2: HTML Lists and Tables

In this lecture, we will learn how to create lists and tables in HTML. Lists help organize data in ordered or unordered ways, and tables let us present data in rows and columns.

1. HTML Lists

An HTML List allows you to organize data on web pages into an ordered or unordered format to make the information easier to read and visually appealing. HTML Lists are very helpful for creating structured, accessible content in web development.

There are 3 types of lists:
  • Unordered List (<ul>)
  • Ordered List (<ol>)
  • Description List (<dl>)

Unordered List (<ul>)

An unordered list is a collection of items where the order does not matter. Each item is marked with a bullet point by default. Use the <ul> tag to create the list, and wrap each item in an <li> tag.

Syntax:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Example:

<ul>
   <li>Bread</li>
   <li>Eggs</li>
   <li>Milk</li>
   <li>Coffee</li>
</ul>

Output:

  • Bread
  • Eggs
  • Milk
  • Coffee

Ordered List (<ol>)

An ordered list is a collection of items where the order matters. Each item is numbered automatically by default. Use the <ol> tag to create the list, and wrap each item in an <li> tag.

Syntax:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Example:

<ol>
   <li>HTML</li>
   <li>CSS</li>
   <li>JavaScript</li>
</ol>

Output:

  1. HTML
  2. CSS
  3. JavaScript

Example with Attributes:

<ol type="i" start="5" reversed>
   <li>HTML</li>
   <li>CSS</li>
   <li>JavaScript</li>
</ol>

Output:

  1. HTML
  2. CSS
  3. JavaScript

Description List (<dl>)

A description list is used to define terms and their descriptions. Use the <dl> tag to create the list. Each term is wrapped in a <dt> tag, and its description is wrapped in a <dd> tag.

Syntax:

<dl>
    <dt>Term 1</dt>
    <dd>Description of Term 1</dd>
    <dt>Term 2</dt>
    <dd>Description of Term 2</dd>
</dl>

Example:

<dl>
   <dt>Coffee</dt>
   <dd>- 500 gms</dd>
   <dt>Milk</dt>
   <dd>- 1 ltr Tetra Pack</dd>
</dl>

Output:

Coffee
- 500 gms
Milk
- 1 ltr Tetra Pack

2. HTML Tables

HTML (HyperText Markup Language) tables are used to display data in rows and columns. Tables help organize information clearly on a web page.

Basic table tags:
  • <table>: Defines the table
  • <tr>: Defines a table row
  • <th>: Defines a table header cell
  • <td>: Defines a table data cell
  • <caption>: Adds a title or caption to the table
  • <thead>: Defines the table header section
  • <tbody>: Defines the table body section
  • <tfoot>: Defines the table footer section

HTML Table (<table>)

An HTML table is used to display data in a row and column format. It is created using the <table> tag. Inside a table, rows are defined with <tr> (table row), header cells with <th>, and data cells with <td>.

Syntax:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Example:

<table border="1">
   <tr>
      <th>Name</th>
      <th>Age</th>
      <th>City</th>
   </tr>
   <tr>
      <td>Kashish</td>
      <td>19</td>
      <td>Patiala</td>
   </tr>
   <tr>
      <td>Arjun</td>
      <td>20</td>
      <td>Delhi</td>
   </tr>
</table>

Output:

Name Age City
Kashish 19 Patiala
Arjun 20 Delhi

Table Row (<tr>)

The <tr> tag in HTML defines a row inside a table. Each row contains one or more <th> (table header) or <td> (table data) cells. You must place <tr> tags inside a <table>.

Syntax:

<table>
    <tr>
        <td>Cell 1</td>
        <td>Cell 2</td>
    </tr>
</table>

Example:

<table border="1">
   <tr>
      <td>Apple</td>
      <td>Red</td>
   </tr>
   <tr>
      <td>Banana</td>
      <td>Yellow</td>
   </tr>
</table>

Output:

Apple Red
Banana Yellow

Table Header (<th>)

The <th> tag in HTML defines a header cell inside a table. Header cells are usually displayed in bold text and centered by default. Use <th> inside a <tr> (table row) to represent column headings.

Syntax:

<table>
    <tr>
        <th>Heading 1</th>
        <th>Heading 2</th>
    </tr>
</table>

Example:

<table border="1">
   <tr>
      <th>Fruit</th>
      <th>Color</th>
   </tr>
   <tr>
      <td>Apple</td>
      <td>Red</td>
   </tr>
   <tr>
      <td>Banana</td>
      <td>Yellow</td>
   </tr>
</table>

Output:

Fruit Color
Apple Red
Banana Yellow

Table Data (<td>)

The <td> tag in HTML defines a standard data cell inside a table. It is used to display the actual content of the table (like text, numbers, or images). Each <td> must be placed inside a <tr> (table row).

Syntax:

<table>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Example:

<table border="1">
   <tr>
      <th>Fruit</th>
      <th>Color</th>
   </tr>
   <tr>
      <td>Apple</td>
      <td>Red</td>
   </tr>
   <tr>
      <td>Banana</td>
      <td>Yellow</td>
   </tr>
</table>

Output:

Fruit Color
Apple Red
Banana Yellow

Table Caption (<caption>)

The <caption> tag is used to add a title or heading for a table. It helps describe the content of the table and usually appears above the table by default. It must be placed immediately after the opening <table> tag.

Syntax:

<table>
   <caption>Table Title</caption>
   <tr>
      <th>Heading 1</th>
      <th>Heading 2</th>
   </tr>
   <tr>
      <td>Data 1</td>
      <td>Data 2</td>
   </tr>
</table>

Example:

<table border="1">
   <caption>Fruit and Their Colors</caption>
   <tr>
      <th>Fruit</th>
      <th>Color</th>
   </tr>
   <tr>
      <td>Apple</td>
      <td>Red</td>
   </tr>
   <tr>
      <td>Banana</td>
      <td>Yellow</td>
   </tr>
</table>

Output:

Fruit and colors
Fruit Color
Apple Red
Banana Yellow

Table Head (<thead>)

The <thead> tag is used to group the header content of a table. It contains one or more <tr> (table rows) with <th> (header cells). It helps keep the table organized and makes it easier for browsers, screen readers, and styling.

Syntax:

<table>
   <thead>
      <tr>
         <th>Heading 1</th>
         <th>Heading 2</th>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>Data 1</td>
         <td>Data 2</td>
      </tr>
   </tbody>
</table>

Example:

<table border="1">
   <caption>Student Marks</caption>
   <thead>
      <tr>
         <th>Name</th>
         <th>Subject</th>
         <th>Marks</th>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>Amit</td>
         <td>Math</td>
         <td>85</td>
      </tr>
      <tr>
         <td>Sita</td>
         <td>Science</td>
         <td>92</td>
      </tr>
   </tbody>
</table>

Output:

Student Marks
Name Subject Marks
Amit Math 85
Sita Science 92

Table Body (<tbody>)

The <tbody> tag is used to group the main content (data rows) of a table. It contains one or more <tr> (table rows) with <td> (data cells). By separating <thead>, <tbody>, and <tfoot>, the table becomes more structured and easier to manage.

Syntax:

<table>
   <thead>...</thead>
   <tbody>
      <tr>
         <td>Data 1</td>
         <td>Data 2</td>
      </tr>
   </tbody>
</table>

Example:

<table border="1">
   <caption>Employee Details</caption>
   <thead>
      <tr>
         <th>Name</th>
         <th>Position</th>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>Ravi</td>
         <td>Developer</td>
      </tr>
      <tr>
         <td>Meera</td>
         <td>Designer</td>
      </tr>
   </tbody>
</table>

Output:

Employee Details
Name Position
Ravi Developer
Meera Designer

Table Footer (<tfoot>)

The <tfoot> tag is used to group the footer content of a table. It usually contains summary rows, totals, or notes about the data. Even though it appears at the bottom of the table visually, it can be placed before <tbody> in the code for faster rendering.

Syntax:

<table>
   <thead>...</thead>
   <tbody>...</tbody>
   <tfoot>
      <tr>
         <td colspan="2">Summary</td>
      </tr>
   </tfoot>
</table>

Example:

<table border="1">
   <caption>Monthly Expenses</caption>
   <thead>
      <tr>
         <th>Category</th>
         <th>Amount</th>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>Food</td>
         <td>$200</td>
      </tr>
      <tr>
         <td>Transport</td>
         <td>$100</td>
      </tr>
   </tbody>
   <tfoot>
      <tr>
         <td>Total</td>
         <td>$300</td>
      </tr>
   </tfoot>
</table>

Output:

Monthly Expenses
Category Amount
Food $200
Transport $100
Total $300
