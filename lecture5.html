Lecture 5: HTML Media and Multi-Page Websites

In this lecture, we will learn how to use audio, video, iframe, and how to create multi-page websites with proper header, main, and footer sections. We will also understand embedding YouTube videos and deploying websites.

1. The <audio> Tag

The <audio> tag is used to embed sound content in a web page. Common formats: MP3, OGG, WAV. By default, audio does not display anything, so we must use the controls attribute.

Syntax:

<audio src="audio.mp3" controls></audio>

2. The <video> Tag

The <video> tag is used to embed videos. Common formats: MP4, WebM, OGG. We use the controls attribute to allow play, pause, and volume control.

Syntax:

<video src="video.mp4" width="400" controls></video>

Example:

3. Embedding YouTube Videos with <iframe>

To embed a YouTube video, we use the <iframe> tag. This allows us to play online videos inside our website.

Syntax:

<iframe width="400" height="250"
    src="https://www.youtube.com/embed/VIDEO_ID"
    frameborder="0" allowfullscreen>
</iframe>

4. Autoplay Feature

We can add autoplay to audio or video so that it plays automatically when the page loads. ⚠️ Some browsers may block autoplay until the user interacts.

Example:

<audio src="song.mp3" autoplay controls></audio>

5. Multi-Page Websites

A multi-page website has multiple HTML files connected using navigation links. Each page can contain a header, main section, and footer.

Structure:

  • index.html → Homepage
  • about.html → About page
  • contact.html → Contact form page

Example Navigation:

<nav>
  <a href="index.html">Home</a> |
  <a href="about.html">About</a> |
  <a href="contact.html">Contact</a>
</nav>

6. Header, Main, and Footer Sections

HTML5 introduced semantic tags:

  • <header> → Top section (logo, navigation, heading)
  • <main> → Main content of the page
  • <footer> → Bottom section (copyright, links)

Example:

<header>
  <h1>My Website</h1>
  <nav>
    <a href="index.html">Home</a>
  </nav>
</header>

<main>
  <p>This is the main content.</p>
</main>

<footer>
  <p>&copy; 2025 My Website</p>
</footer>

7. Combining iframes with Tables

We can insert an <iframe> inside a table cell for layout purposes.

Example:

<table border="1">
  <tr>
    <td>YouTube Video</td>
    <td>
      <iframe width="300" height="200"
        src="https://www.youtube.com/embed/tgbNymZ7vqY"
        allowfullscreen>
      </iframe>
    </td>
  </tr>
</table>

8. Deploying Website on Netlify

Netlify is a free hosting platform for deploying websites. Steps to deploy:

  1. Create an account on Netlify.
  2. Upload your project folder (HTML, CSS, JS files).
  3. Netlify generates a live URL for your website.
