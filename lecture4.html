HTML Forms

HTML forms, defined using the Tags are essential for collecting user input on web pages. They incorporate a variety of interactive controls such as text fields, numeric inputs, email fields, password fields, checkboxes, radio buttons, and submit buttons. Over 85% of websites rely on forms to gather data from users, making them a fundamental component of modern web development.

Key points:
  • Forms are created using the <form> tag.
  • Form elements are added inside the form, such as <input>, <textarea>, <select>, and <button>.
  • The action attribute specifies the server URL to send data to.
  • The method attribute specifies how data is sent: GET or POST.

1. Basic Form Structure

A basic HTML form uses the <form> tag with action and method attributes.

Syntax:

<form action="submit.php" method="post">
    Form elements go here
</form>

Example:

<form action="/submit.php" method="post">
    <label>Name:</label>
    <input type="text" name="username"><br><br>

    <label>Email:</label>
    <input type="email" name="email"><br><br>

    <input type="submit" value="Submit">
</form>

HTML Form Elements

Element Description
<label> It defines labels for <form> elements.
<input> It is used to get input data from various types such as text, password, email, etc., by changing its type.
<button> It defines a clickable button to control other elements or execute a functionality.
<select> It is used to create a drop-down list.
<textarea> It is used to get input long text content.
<fieldset> It is used to draw a box around other form elements and group the related data.
<legend> It defines a caption for fieldset elements.
<datalist> It is used to specify pre-defined list options for input controls.
<output> It displays the output of performed calculations.
<option> It is used to define options in a drop-down list.
<optgroup> It is used to define group-related options in a drop-down list.

Input Types

HTML forms support many types of input elements. Some common ones:

  • text ‚Äì single-line text input
  • password ‚Äì hidden text for passwords
  • email ‚Äì input for email addresses
  • number ‚Äì numeric input
  • radio ‚Äì select one option
  • checkbox ‚Äì select multiple options
  • submit ‚Äì send the form data
  • reset ‚Äì reset all fields to default values

Example with multiple input types:

<form action="/submit.php" method="post">
    <label>Username:</label>
    <input type="text" name="username"><br><br>

    <label>Password:</label>
    <input type="password" name="password"><br><br>

    <label>Gender:</label>
    <input type="radio" name="gender" value="male"> Male
    <input type="radio" name="gender" value="female"> Female<br><br>

    <label>Hobbies:</label>
    <input type="checkbox" name="hobby1" value="reading"> Reading
    <input type="checkbox" name="hobby2" value="sports"> Sports<br><br>

    <input type="submit" value="Submit">
</form>

Male Female

Reading Sports

HTML Input Types

Input Type Description
<input type="text"> Defines a one-line text input field.
<input type="password"> Defines a password field.
<input type="submit"> Defines a submit button.
<input type="reset"> Defines a reset button.
<input type="radio"> Defines a radio button.
<input type="email"> Validates that the input is a valid email address.
<input type="number"> Allows the user to enter a number. You can specify min, max, and step attributes for range.
<input type="checkbox"> Used for checkboxes where the user can select multiple options.
<input type="date"> Allows the user to select a date from a calendar.
<input type="time"> Allows the user to select a time.
<input type="file"> Allows the user to select a file to upload.

Task: Create a Registration Form

Create a simple registration form for a website that collects the following information from the user:

  • Full Name (text input)
  • Email Address (email input)
  • Password (password input)
  • Gender (radio buttons: Male, Female, Other)
  • Hobbies (checkboxes: Reading, Sports, Music, Traveling)
  • Country (dropdown select)
  • About Yourself (textarea)
  • Submit Button

Hints:

  • Use <form> to wrap your inputs.
  • Use <label> for each input field for better accessibility.
  • Use name attributes for all inputs to identify the data.
  • Use <fieldset> and <legend> to group related inputs if you want.
