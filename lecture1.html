Lecture 1: Basic HTML Tags

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: Links that connect web pages to each other.
  • Markup Language: Uses tags to ‚Äúmark up‚Äù content with meaning and structure.

1. Headings

Headings define titles and subtitles on your page, from <h1> to <h6>.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
...
<h6>Heading 6</h6>

2. Paragraphs

<p>This is the first paragraph.</p>
<p>This is another paragraph.</p>

3. Horizontal Rule

<p>Topic One</p>
<hr>
<p>Topic Two</p>

4. Line Break

<p>221B Baker Street<br>London, England</p>

5. Lists

Unordered list:

<ul>
  <li>Tea Bag</li>
  <li>Water</li>
</ul>

Ordered list:

<ol>
  <li>Boil water</li>
  <li>Add tea bag</li>
</ol>

6. Anchor Tag

<a href="https://www.coderarmy.in/" target="_blank">Visit Coder Army</a>

7. Image Tag

The <img> tag embeds images. Attributes: src (path/URL) and alt (description).

<img src="https://cdn.britannica.com/16/234216-050-C66F8665/beagle-hound-dog.jpg"
     alt="Beagle Dog" width="400">
